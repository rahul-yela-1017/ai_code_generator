# AI Code Generator & Explainer

An AI-powered application that transforms natural language prompts into executable code and provides clear, step-by-step explanations of the generated output.

## ✨ Features

* Generate code from plain English prompts
* Supports Python, Java, and C++
* AI-powered code explanations
* Debugging suggestions and improvements
* Save generated code locally
* Interactive Streamlit interface

## 🛠️ Tech Stack

* Python
* Streamlit
* LangChain
* Ollama
* CodeLlama / DeepSeek-Coder

## 🚀 Getting Started

```bash
git clone <repository-url>
cd AI-Code-Generator-Explainer
pip install -r requirements.txt
streamlit run app.py
```

## 🎯 Purpose

This project demonstrates how Generative AI can simplify software development by converting human language into functional code while enhancing learning through intelligent explanations.
